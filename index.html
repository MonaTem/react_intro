Introduction to React

What is React?

React is a JavaScript library that focuses on user interfaces. React is NOT a framework. You can use React with other libraries and frameworks, for example Bootstrap or Materialize.

Who made it?

React was developed by Jordan Walke, a software engineer at Facebook. The first deployment was with Facebook's newsfeed in 2011. It is maintained by developers and Facebook and Instagram.

The first public release was in March 2013

What's a user interface?

The screens, pages, and visual elements (buttons, icons, menus) that you use to interact with a device or application.

Why use React?

Usage of JSX makes it easier to template

React Native makes it easier to create mobile apps

Makes use of a Virtual DOM which makes DOM manipulation faster

Allows you to create customizable elements that you can easily reuse
